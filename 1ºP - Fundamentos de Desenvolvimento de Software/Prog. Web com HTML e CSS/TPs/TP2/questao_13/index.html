Seletores CSS

Vamos falar de seletores! Vou te mostrar como eles funcionam de forma fácil e bonitinha, bora lá?

Esse aqui é um parágrafo básico, ele está sendo estilizado usando o seletor por tag. Tudo que for um <p> nesse documento vai seguir essas regras.

Agora temos um parágrafo com uma classe chamada "destaque". Isso quer dizer que qualquer elemento que usar essa classe vai receber essa mesma estilização.

Esse aqui é um parágrafo normal, mas logo abaixo tem um link:

Clique aqui!

Esse link usa um seletor de atributo, que estiliza qualquer link que tenha um atributo "href". Ah, e quando você passar o mouse em cima dele, ele muda! Isso é o seletor de pseudo-classe :hover em ação.

Aqui temos um exemplo de seletor descendente. Esse parágrafo está dentro de uma <div> e, por isso, ele tem um estilo diferente dos outros parágrafos fora da <div>.

Resumindo, temos então 6 tipos apresentados:

  • Seletor por tags
  • Seletor por classe
  • Seletor por id
  • Seletor de atributo
  • Seletor de pseudo-classe
  • Seletor descendente
